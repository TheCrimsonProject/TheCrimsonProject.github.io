The Crimson Project :: プログラミング言語の実験場

About

"Why can't languages themselves be the artwork?"
- Chariette Hsu, 2018

Crimson is a family of unrelated languages designed by the same person committing to the same aesthetic.

The Crimson Project started as an attempt to design and implement a purely functional dialect of R4RS Scheme w/o macros, which is in turn to replace Racket in a commercial codebase. The original project itself was postponed, yet the spirit of it lives on and fuses with another core idea (written above as a quote) of the project.

Crimson currently consists of 3 languages (1 discontinued), which are all in early-design stage.
